# LUMINO

> *Reveal the light hidden in the grid*

A minimal, handcrafted **Picross / Nonogram** puzzle game — built as a single HTML file with no dependencies, no framework, no build step.

Paint cells by cracking row and column number clues. Solve the grid. Reveal the hidden pixel art.

**[▶ Play it live](https://aibuilt-apps.github.io/LUMINO_Game)**

---

## How to play

Each row and column has a sequence of numbers.
Each number represents a consecutive block of filled cells — in order, with at least one gap between blocks.

| Action | Control |
|---|---|
| Fill a cell | Left click |
| Mark empty (×) | Right click |
| Clear a cell | Click again |
| Get a hint | Hint button (3 per puzzle) |

Clue numbers grey out automatically when a line is correctly completed.
The puzzle is solved when every row and column matches its clues exactly.
